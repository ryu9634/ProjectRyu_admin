취소처리 도움말

취소처리 개요

참고: 관리자(판매자)가 주문관리에서 판매취소 처리를 해도 취소요청으로 접수되어 취소승인 과정이 필요합니다.

취소승인

취소요청 상태의 주문을 환불 처리할 때 취소승인 버튼을 누릅니다. 자동환불이 불가한 경우 추가 절차가 필요합니다.

자동 환불이 가능한 경우

자동환불이 가능한 주문은 판매자께서 신경쓰지 않아도 취소승인시 2~5영업일 이내에 처리됩니다.(카드사에 따라 소요일이 다를 수 있습니다.)

  • 신용카드 주문
  • 가상계좌 주문 (이지페이, 이니시스 : 자동환불 가능. 이니시스 가맹점 중 국내 전자결제 설정에서 가상계좌 자동환불 사용 설정된 경우에만 사용 가능합니다.)
  • 실시간 계좌이체 주문
  • 휴대폰 결제 주문 (이지페이 : 자동환불 가능)
  • 네이버페이 주문
  • PAYCO 주문 (KCP)
  • 카카오페이 주문 (이니시스)
  • 엑심베이 주문 (해외신용카드, Alipay, WeChat, eContext)

자동 환불이 불가한 경우

자동환불이 불가한 주문은 취소승인 후 취소처리중 상태가 되며, 구매자가 신청한 환불방법 또는 구매자와 환불방법을 협의해 실제 환불 후 기타 취소처리 > 취소(환불) 완료 처리 해주셔야 합니다.

  • 무통장입금 주문
  • 가상계좌 주문 (KCP, 나이스페이 : 자동환불 불가)
  • 휴대폰 결제 주문 (KCP, 이니시스, 나이스페이 : 현재월 결제에 한해 PG사 관리페이지에서 취소처리 가능)
  • 카카오페이 구매 주문 (카카오페이 구매 파트너 센터에서 취소처리 가능)
  • PayPal 주문 (PayPal 판매자 페이지에서 취소처리 가능)
  • MOLPay 주문 (취소처리 불가. MOLPay에서 취소, 환불 처리 미지원. 말레이시아 통화 한정으로 고객이 직접 몰페이에 환불접수/처리 가능)

네이버페이 부분취소시 환불처리 시점

예를들어 3개의 품목주문 취소요청 중 1개의 주문만 취소승인시 환불보류 상태가 되며, 나머지 2건의 품목주문이 모두 처리(취소 또는 배송) 되었을 때 환불이 진행됩니다.

요컨대 주문번호 내에 미처리 품목주문이 1개라도 존재하면 환불보류 상태가 되어 환불이 진행되지 않습니다.

신용카드 부분취소시 환불처리 시점

네이버페이 부분취소와 달리 일부 미처리 취소요청 품목주문이 있더라도 취소 승인한 품목주문의 금액만큼 환불이 진행됩니다.

취소거절

취소요청을 거절할 수 있습니다. 취소요청 거설시 발송처리됩니다.

취소거절시 배송방법에 따라 배송정보를 입력해야 합니다.

  • 택배,소포,등기: 발송처리를 위해 배송정보(택배사, 운송장번호)를 입력해야 합니다.
  • 배송없음, 디지털상품, 퀵배달, 방문수령: 배송정보를 입력할 필요가 없습니다.

기타 취소처리 - 되돌리기

기타 취소처리의 ~되돌리기는 이전 단계로 되돌리는 기능입니다. 이후 단계로 처리는 불가합니다.(예: 취소처리중에서 취소요청으로 되돌리기(가능) / 취소요청에서 취소요청으로 되돌리기(불가)

네이버페이 주문은 이전 단계로 되돌리는 기능을 네이버 정책상 지원하지 않습니다.

  • 취소처리중으로 되돌리기: 취소완료건에 대해 취소처리중으로 되돌립니다. 이미 환불한 경우 환불 대금이 회수되진 않습니다.
  • 취소요청으로 되돌리기: 취소완료, 취소처리중을 취소요청으로 되돌립니다.

기타 취소처리 - 자동환불 재시도

네이버페이를 제외하고 자동환불이 가능한 주문의 취소승인시 PG사 또는 네트워크 오류 등으로 자동환불 처리가 되지 않을 수 있습니다.

자동환불 처리가 실패하면 취소처리중(자동환불 처리실패) 상태로 표시되며 이 때 자동환불 재시도 버튼을 사용할 수 있습니다.

만약 계속해서 자동환불 실패가 발생할 경우, 구매자와 환불 방법을 협의해 환불처리 후 기타 취소 처리 > 취소(환불)완료 처리해 주시기 바랍니다.

자동환불 실패가 발생할 수 있는 결제수단

  • 신용카드
  • PAYCO
  • 가상계좌

기타 취소처리 - 취소(환불)완료 처리

자동환불이 지원되지 않는 취소처리중 주문 또는 자동환불 실패가 발생한 주문을 강제로 취소완료 처리하는 기능입니다. 실제 환불이 되지 않으므로 이 작업 이전에 구매자에게 환불처리 이후 진행해 주시기 바랍니다.

자동환불이 지원되지 않는 상황

  • 무통장입금 주문
  • 가상계좌 주문 (KCP, 나이스페이 : 자동환불 불가)
  • 휴대폰 결제 주문 (KCP, 이니시스, 나이스페이 : 현재월 결제에 한해 PG사 관리페이지에서 취소처리 가능)

  • PayPal 주문 (PayPal 판매자 페이지에서 취소처리 가능)
