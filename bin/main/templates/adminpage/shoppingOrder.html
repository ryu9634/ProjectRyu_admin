주문관리 개요

주문 선택하기

주문을 처리하기 위해서는 먼저 처리할 주문을 체크해야 합니다.

  • 주문번호: 주문 단위에 부여되는 고유의 번호입니다. 장바구니에 여러 상품을 넣고 한 번에 주문하면 하나의 주문번호로 묶여 표시됩니다. 주문번호를 체크하면 해당 주문의 품목주문번호도 함께 선택됩니다.

    일반적인 상황에서는 주문번호 단위로 선택하여 처리하는게 편리합니다.

  • 품목주문번호: 상품이나 옵션마다 부여되는 고유 번호입니다. 여러 상품을 한 번에 주문하면 각각의 상품마다 품목주문번호가 부여됩니다.

    같은 주문번호 내 특정 품목주문만 판매를 취소하거나, 재고가 부족해 별도의 처리가 필요한 경우 품목주문을 선택해 처리할 수 있습니다.

발송 처리하기

선택 주문에 대해 발송(배송)처리합니다.
발송처리 가능 상태: 결제완료, 배송준비중

배송방법에 따른 발송처리 안내

  • 택배,소포,등기
    • 발송처리 후 상태: 배송중
    • 필수 항목: 택배사, 운송장번호
    • 기타택배 및 배송추적이 불가능한 택배사는 발송 처리 후 10일이 지나면 자동 배송 완료가 됩니다. 10일 전에 배송 완료를 마쳤다면, 기타 주문처리 > 강제 배송 완료 처리를 해주셔야 합니다. (네이버페이 주문엔 적용되지 않습니다.)
    • 네이버페이 주문을 기타택배로 발송시 구매자가 별도 구매확정을 하지 않는한 네이버 정책에 의해 발송일로부터 28일째 되는 날 자동구매확정 되면서 배송완료 처리됩니다.

  • 직접배송, 방문수령, 퀵배달
    • 발송처리 후 상태: 배송중
    • 직접배송, 방문수령, 퀵배달은 발송 처리 후 10일이 지나면 자동 배송 완료가 됩니다. 10일 전에 배송 완료를 마쳤다면, 기타 주문처리 > 강제 배송 완료 처리를 해주셔야 합니다. (네이버페이 주문엔 적용되지 않습니다.)

  • 배송없음
    • 발송처리 후 상태: 배송완료
    • 배송없음 주문은 주로 무형의 상품(쿠폰, 티켓, 입장권 등)을 SMS, 메일 및 기타 수단으로 전달하기 때문에 발송처리 후 배송중 상태를 거치지 않고 배송완료 처리됩니다.

  • 디지털 다운로드 상품
    • 디지털 다운로드 상품은 입금확인 또는 결제완료시 배송완료 처리되기 때문에 발송처리를 거칠 필요가 없습니다.

배송준비 처리

결제완료 또는 입금확인 주문에 대해 상품을 준비하는 과정에서 구매자가 임의로 취소할 수 없도록 설정하는 기능입니다.

  • 배송준비 처리 후 상태: 배송준비중
  • 배송 준비중으로 설정된 주문을 구매자가 취소(환불)하려면 운영자의 승인이 필요합니다.
  • 배송 준비중 주문의 구매자 취소요청 거부시 발송처리됩니다.(배송정보 입력 필요)
  • 배송준비 처리는 선택사항이며 결제완료 단계에서 배송중으로 처리할 수 있습니다.

입금확인 처리

무통장입금 방식으로 결제한 고객이 대금 입금을 확인한 경우 처리합니다.

  • 입금확인 처리 후 상태: 결제완료
  • 입금대기 상태의 주문은 구매자가 언제든지 취소할 수 있습니다.

기타주문 처리 - 되돌리기

기타 주문처리의 ~되돌리기는 이전 단계로 되돌리는 기능입니다. 이후 단계로 처리는 불가합니다.(예: 배송중에서 배송준비로 되돌리기(가능) / 입금대기에서 결제완료로 되돌리기(불가)

네이버페이 주문은 이전 단계로 되돌리는 기능을 네이버 정책상 지원하지 않습니다.

  • 입금대기로 되돌리기: 결제완료인 무통장입금 주문을 입금대기 상태로 되돌립니다.
  • 결제완료로 되돌리기: 배송준비중, 배송중 상태의 주문을 결제완료 상태로 되돌립니다.
  • 배송준비로 되돌리기: 배송중, 배송완료 상태의 주문을 배송준비중 상태로 되돌립니다.
  • 배송중으로 되돌리기: 배송완료 상태의 주문을 배송중 상태로 되돌립니다.

기타주문 처리 - 강제 배송완료 처리

배송추적이 지원되지 않는 배송방법(기타택배, 직접배송, 방문수령, 퀵서비스)을 사용한 주문을 배송완료 처리시 사용합니다.

네이버페이 주문은 강제 배송완료 처리를 네이버 정책상 지원하지 않습니다. 배송추적이 불가한 주문은 발송일로부터 28일 후 자동 구매확정이 되면서 자동 처리됩니다.

  • 배송추적이 지원되는 배송방식에서도 사용 가능합니다.
  • 배송중이 아닌 상태(입금대기, 결제완료 등)에서도 사용 가능합니다.

기타주문 처리 - 발송지연 처리(Npay용)

이 기능은 네이버페이 주문만 해당됩니다.

  • 네이버페이 정책상 결제완료일로부터 3영업일 이내 미발송시 패널티가 부과됩니다.
  • 패널티 부과를 피하려면 발송지연 처리를 해주셔야 합니다.
  • 발송지연처리는 주문당 1회에 한해서만 가능하며 발송기한 경과시까지 발송되지 않으면 구매자 취소 요청시 즉시 환불됩니다.

판매취소 처리

판매자께서 강제로 판매를 취소 또는 환불하고자 할 때 사용하는 기능입니다. 디지털 상품, 배송없음 상품은 반품/교환 요청 및 처리가 불가하며 취소를 통해 환불을 진행할 수 있습니다.

취소 및 환불 처리방법

  1. 판매취소를 하면 위와 같이 판매취소사유를 선택 후 확인을 누르면 취소요청 상태로 변경됩니다.
  2. 취소요청 상태의 주문 관리는 취소요청관리에서 처리 가능합니다.
  3. 취소요청관리에서 주문을 선택 후 취소 승인을 누릅니다.
  4. 자동 환불이 가능한 상황(신용카드, 실시간 계좌이체, 네이버페이 주문, 해외신용카드, AliPay, WeChatPay, eContext)에서는 취소 승인 후 자동으로 환불 처리됩니다.(영업일 기준 2~3일 소요)
  5. 자동 환불이 불가한 상황(가상계좌, 휴대폰결제, 무통장입금, MOLPay)은 취소 승인 후 취소 처리 중상태가 됩니다.
    • 구매자에게 환불을 완료하였으면 기타 취소처리 > 취소(환불)완료 처리를 선택하여 취소를 완료합니다.

반품 처리

배송이 완료된 주문을 구매자가 환불을 원할 때 사용합니다.

네이버페이 주문은 판매자가 반품처리를 진행하는 경우에는 자동수거가 진행되지 않고, 직접수거하거나 택배사를 통해 수거요청을 해주셔야 하며, 반품비용 처리방법은 판매자의 계좌로 송금하는 방법으로만 제공이 가능합니다.

반품 처리 방법

  1. 반품처리 버튼을 누른 후 반품 사유와 반품 수거정보를 입력하고 확인울 누르면 네이버페이 주문의 경우 수거진행중으로 처리되며 일반 주문의 경우 반품요청으로 전환 됩니다.
  2. 이후부터는 반품관리에서 진행합니다.
